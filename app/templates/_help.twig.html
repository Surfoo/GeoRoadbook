Need help?

Your roadbook is now displayed in text editor mode.
You can change, delete, move, add contents, insert images or even create new pages.

Action buttons

  • Save: Save your roadbook on the server. You can know the last save date with a hover on this button.
  • Export as PDF: Create your roadbook in PDF, some options are available: format, margins, and a custom header or footer.
  • Download as: This button offers the following options in a drop down list:
    • HTML: Your roadbook will be displayed in your web browser, like a normal web page.
    • ZIP: A dialog box will be displayed in which you can decide where to save the zipped HTML roadbook and rename the file.
    • PDF: If you have previously clicked the "Export as PDF" button, this third option is offered. A dialog box will be displayed in which you can decide where to save the roadbook in PDF format and rename the file.
  • Delete: When you click this button your roadbook will be destroyed. There is no undo for this function.
